Sample Challenges

Each month tackles a different Linux domain.

Month 1

🔧 Broken Apache

A web server stopped after a config change. Find the typo, fix it, bring it back online.

Month 2

🔐 Permission Denied

A deployment script refuses to run. Diagnose permissions, fix it without chmod 777.

Month 3

🌐 Network Nightmare

Server can't reach the internet. DNS, routes, firewall — debug the stack.

Month 4

⚙️ Systemd Strikes

A service won't start. Unit file errors, dependencies, socket activation — master systemd.

Month 5

💾 Disk Disaster

Disk says full but du shows free space. Find the deleted files still held open.

Month 6

🔥 Firewall Fiasco

App works locally but not remotely. iptables, ufw, nftables — which one is blocking?
